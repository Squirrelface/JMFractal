About this site

This site was created as a high school project. The overall purpose of the project was to investigate a hypothesis and write a report of the findings.

We wanted to investigate if there was a significant difference in different browsers speed for simple mathematics. To make the project more fun we decided to investigate different browsers speed when calculting values for the Mandelbrot set. These calculations consist of simple multiplications, additions, subtractions, assignement and comparisons.

Source code and results

The source code and report (in Swedish) can be found at github repo.

A brief summry of the results can be found in this reddit post.

Usage

The user can generate a Mandelbrot by entering values for size of image, which part of the complex plane that should be rendered and which color scheme to use. Alternatively, the user can click and drag on an area of the image to zoom in there.

Create a desktop background

If you want a beutifull desktop image just enter the dimensions of your desktop in the width and height and adjust the rest of the parameters to your liking. When you have an image you want just right click the image and choose "Save as...".
